# e-commerce
Simple e-commerce application on microservice architecture for learning purpose

microservices and tech stack:
1. customer - Spring
2. product - Servlet
3. order - Spring
4. rating - SpringBoot

Major focus is on backend learning so for UI, JSP is used.
 
Spring libraries used:
1. Eureka
2. Config
3. Api Gateway

Tools:
1. OpenJDK 17
